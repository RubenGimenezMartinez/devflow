# Knowledge Base

Base de conocimiento gestionada automáticamente por DevFlow.

Cada archivo `.md` representa una categoría y se actualiza tras cada análisis de issues.

## Categorías

- **business-rules.md** — Reglas de negocio descubiertas
- **dependencies.md** — Dependencias y conexiones entre componentes
- **configuration.md** — Configuración del proyecto
- **patterns.md** — Patrones de código y arquitectura
- **bugs.md** — Bugs conocidos, causas raíz y soluciones
